# Student Registration System
A fully responsive, web-based Student Registration System built using HTML5, CSS3, and Vanilla JavaScript. This project allows users to register student details, manage records through CRUD operations, and ensures data persistence using Local Storage.

## 🚀 Features

### 1. Registration Form (Task 3)
    - Captures Student Name, Student ID, Email ID, and Contact Number.
    - User-friendly layout with clear labels and placeholders.
### 2. Records Display (Task 4)
    - A dedicated section to view all registered students in an organized table format.
    - Action buttons for Editing and Deleting records.
    - Dynamic Scrollbar: A vertical scrollbar appears automatically via JavaScript when the list exceeds 5 records to maintain layout integrity.
### 3. Input Validation (Task 6)
    - Strict validation logic to ensure data integrity:
    - Student Name: Accepts only alphabetic characters and spaces.
    - Student ID: Accepts only numerical values.
    - Email: Validates proper email format (e.g., name@domain.com).
    - Contact Number: Accepts only numbers and requires at least 10 digits.
    - Empty Rows: Prevents submission of empty or incomplete fields.
### 4. Data Persistence (Task 6)
    - Implements Local Storage so that student records remain available even after refreshing the browser page or closing the window.
### 5. Responsive Design (Task 5)
    - The system is fully optimized for three major screen sizes:
    - Mobile: (≤ 640px) - Stacked layout for easy scrolling.
    - Tablet: (641px – 1024px) - Balanced layout adjustments.
    - Desktop: (≥ 1025px) - Side-by-side view of the form and display table.

## 🛠️ Technologies Used

### HTML5: Semantic structure and form elements.
### CSS3: Custom styling, Flexbox/Grid, and Media Queries for responsiveness.
### JavaScript (ES6): DOM manipulation, CRUD logic, Regex validation, and Local Storage API.
